# Gallery

# Installation and usage
<h2>1. cd to the downloaded and extracted repo folder.</h2>
<h2>2. Create a virtual environment and install the requirements:</h2>
<code>
python -m venv env
.\env\Scripts\activate
pip install -r requirements.txt
</code>
<h2>3. To run:</h2>
<code>
python gallery.py
</code>
